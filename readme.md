# Z-Uno zw8051: development platform for Platform IO

## Usage

Install platform IO
Create platform IO project and configure a platform option in platformio.ini file:

### Stable Version

[env:stable]
platform = zw8051
board = ....
...

### Development Version

[env:development]
platform = https://github.com/andyb749/platform-zw8051-git
board = ...
...
